# CSGO Demo Batch

批量收集CSGO比赛、赛事等信息。

## 说明

利用 node `hltv` 库获取 Match 和 Event 的信息，保存 raw json 文件，另保存 Match: id - download_id - download_url - filename 和 Event: id - name - logo_url - match_list。
